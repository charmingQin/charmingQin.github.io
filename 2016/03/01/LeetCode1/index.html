LeetCode第一题:twoSum问题

［算法］

去大公司面试的同学经常会被问到算法的题目,而在leetcode上刷题也成了很好的习惯,笔者也开始尽量一天解决一道算法题.

现在就开始leetcode的第一题twoSum问题.(笔者用C语言解决)

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,

return [0, 1].

翻译过来,给定一个整型数组, 返回两个下标, 这两个下标对应的数组值相加能够得到taget.

先用最简单,最暴力的方法,直接遍历相加,等于taget就获取到index值.

 /**
 * Note: The returned array must be malloced, assume caller calls free().
*/
int* twoSum(int* nums, int numsSize, int target) {
 int *tempArr = (int*)malloc(2 * sizeof (int));
  for(int i=0; i<numsSize-1; i++) {
      for(int j=i+1; j<numsSize; j++){
          if(nums[i] + nums[j] == target){
              tempArr[0] = i;
              tempArr[1] = j;
          return tempArr;
       }
     }
   }
  free(tempArr);
  return NULL;
}
